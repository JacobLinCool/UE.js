# UE.js - User Exists?
A JavaScript API to check if the user exists.

## Support Sites
1. Instagram
2. Github

## Usage

```javascript
ue.instagram(USERNAME).then(r=>console.log(r)); // true or false

// simpler way
ue.ig(USERNAME).then(r=>console.log(r)); // true or false
```

### Keywords

Instagram: instagram, ig

Github: github, gh
